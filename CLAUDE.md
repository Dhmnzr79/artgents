# artgents

Простой сайт: HTML + CSS + vanilla JS. Без фреймворков и препроцессоров.

## HTML

- Семантические теги по смыслу: `header`, `main`, `nav`, `section`, `footer`.
- Кликабельные элементы — только `<button type="button">`, не `div`/`span`.
- Без inline-обработчиков (`onclick` и т.п.) — слушатели в JS-файле.
- Не плодить лишние обёртки.

## Дизайн-система

- Макет широкий: контейнер `min(1760px, ...)`, расчёт под 1600–1900px экраны.
- Сетка 12 колонок, `gap: var(--gap)`.
- Тёмные секции — чёрно-фиолетовый фон с едва заметными пятнами градиента.
- Светлые секции — светло-серый с такими же едва заметными градиентами.
- Акценты: фиолетовый `#7b6ef6` (UI, кнопки) + бирюзовый `#2dd4bf` (highlights, иконки).
- Зелёный `#4ade80` — только для статусов (онлайн-точка, галочки), не дизайн-акцент.
- Мини-заголовки секций — `.label` с пульсирующей точкой `::before`. Сохраняются везде.
- Навигация — glassmorphism: логотип + 3 пункта меню + кнопка CTA.

## CSS

- Без `style=""` и `!important`.
- Именование классов — BEM.
- Цвета, отступы, радиусы, шрифты — через CSS-переменные в `:root`.
- Селекторы не глубже 3 уровней.
- Media queries — mobile-first (`min-width`), одним блоком в конце файла.

## JS

- Без `console.log` в финальном коде.
- Без закомментированного и мёртвого кода.
- `querySelector` — локально в своём модуле, не разбросан по проекту.
